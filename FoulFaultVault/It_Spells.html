🔡 v5.4 Pure Semantic Parser

No dictionary – every word is parsed letter by letter. Consonants = primitives, vowels = operators between them.

🔹 Consonants (primitives)

B = body / root
C = container / shape_curved
D = durable / object
F = flow / fluid
G = ground / base
H = holding / enclosure
K = cut / separation / stop
L = layer / level
M = motion / mass
N = negation / not
P = person / point
Q = question / query
R = repeat / again
S = state / stillness
T = tool / touch / straight
V = view / vision
W = wave / water
X = negation / hide
Y = why / question
Z = end / final

🔸 Vowels (operators)

A = UNION – merge meanings (siblings)
E = SEQUENCE – first then second (temporal)
I = PROPERTY – second is attribute of first
O = ALTERNATIVE – xor (either left or right)
U = CONTAINER – second inside first

📐 How parsing works

1. Split word into letters (case‑insensitive).
2. Identify consonants (C) and vowels (V).
3. Starting from left, combine C V C → apply operator to left and right.
4. Result becomes new left, continue if another vowel+consonant follows.
5. Extra vowels without a following consonant are ignored (warning).
6. Consecutive consonants without vowel are treated as separate primitives (no operator).
